# AI 학습 플래너 (AI Learning Planner)

사용자가 학습 목표, 학습 수준, 학습 기간을 입력하면 AI가 맞춤형 학습 커리큘럼을 자동으로 생성해주는 웹 사이트


# 주요 기능:
1. 학습 목표 입력 → AI가 세부 질문 생성 (/api/generate)
2. 질문에 답변하면 맞춤 커리큘럼 생성 (/api/build)
3. 채팅으로 커리큘럼 수정 요청 (/api/chat)
4. 마크다운 렌더링, 클립보드 복사, PDF 내보내기


# 기술 스택:
## 프론트 (Frontend)
- React, Tailwind CSS 4

## 백엔드 (Backend)
- Java21 / SpringBoot 3.5.0 / Docker / Upstage Solar Pro

# 회의록 샘플 - 데모 준비 회의

## 붙여넣기용 원문

2026년 6월 7일 데모 준비 회의록

참석자: 박성종, 이동근, 이우태
작성자: 박성종

오늘 회의에서는 6월 11일 발표 전에 남은 작업과 리허설 일정을 정리했다. 먼저 6-1 분석 단계는 Solar API를 실제로 연결해서 시연하기로 했다. 박성종은 6월 10일 오후 6시까지 6-1 분석 흐름과 confidence 산정 기준을 발표 자료에 정리한다. 이동근은 같은 날 오후 6시까지 6-2 라우팅, 충돌 검증, 저장소 흐름을 다시 점검한다. 이우태는 6월 10일 오후 9시까지 프론트엔드 화면 흐름과 저장소 탭 표시를 최종 확인한다.

전체 리허설은 2026년 6월 11일 오전 10시에 진행하기로 했다. 리허설에서는 미니PC에서 백엔드와 프론트엔드를 모두 띄우고, 노트북에서는 Tailscale 주소로 접속해서 입력부터 저장소 확인까지 한 번에 진행한다.

리스크도 몇 가지 확인했다. 미니PC 접속이 불안정하면 노트북 로컬 환경으로 전환한다. Solar API 호출이 실패하면 같은 입력으로 다시 시도하되, 발표 중에는 FakeLLM 대체 흐름도 설명할 수 있도록 준비한다. CORS 설정이 빠지면 노트북 브라우저에서 API 요청이 실패할 수 있으므로 발표 전 환경변수를 다시 확인한다.

추가 메모로, 발표 자료에는 실제 외부 캘린더 연동은 하지 않고 SQLite 저장소를 사용한다는 점을 명확히 적기로 했다.

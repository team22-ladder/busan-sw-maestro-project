# Go-To-Market 전략 (GTM Strategy)

## 개요
좋은 제품은 필요조건이지 충분조건이 아니다. PMF를 향해 나아가는 과정에서 "어떻게 첫 고객을 확보하고, 어떤 채널로 성장할 것인가"에 대한 명확한 전략이 없으면 아무리 훌륭한 AI 기능도 쓸모없다. AI 스타트업의 GTM은 기술이 아닌 고객 접점 설계 문제다.

---

## 1. B2C AI 서비스 GTM 채널표

| 채널 | 특성 | 초기 비용 | 효과 발현 시점 | 적합한 단계 |
|---|---|---|---|---|
| **커뮤니티 마케팅** | 네이버 카페, 카카오오픈채팅, Reddit 등 직접 참여 | 거의 없음 | 즉시~2주 | 초기 검증 |
| **콘텐츠/SEO** | 블로그, 유튜브, 뉴스레터로 유기적 트래픽 | 시간 비용 | 3~6개월 | 제품 안정 후 |
| **Product Hunt / 런칭 플랫폼** | 글로벌 얼리어답터 집중 노출 | 거의 없음 | 런칭일 1~3일 | MVP 출시 시 |
| **인플루언서/크리에이터** | 타겟 크리에이터 협업, 바이럴 잠재력 | 중간 | 1~4주 | PMF 진입 후 |
| **유료 광고 (Meta/Google)** | 즉각적 트래픽, 높은 CAC | 높음 | 즉시 | PMF 확인 후 |
| **파트너십** | 이미 타겟 고객을 가진 플랫폼/기업과 협업 | 협상 비용 | 1~3개월 | 성장 단계 |
| **입소문(WoM)** | 사용자가 자발적으로 추천 | 없음 | PMF 이후 | PMF 달성 후 |

## 2. 첫 100명 전략 3단계

```
1단계: 0→10명 (직접 영업, 수작업)
├─ 방법: 창업자가 직접 타겟 커뮤니티에서 1:1 접근
├─ 채널: LinkedIn DM, 오픈채팅 콜드 메시지, 지인 소개
├─ 목표: 제품이 아닌 문제를 이야기하며 관계 형성
└─ 성공 기준: 10명이 스스로 매일 사용

2단계: 10→50명 (채널 테스트)
├─ 방법: 2~3개 채널에서 동시 소규모 테스트 (각 채널 $0~100 투자)
├─ 채널: 커뮤니티 포스팅, Product Hunt, 틈새 뉴스레터 광고
├─ 목표: 채널별 전환률과 리텐션 비교
└─ 성공 기준: 가장 효율적인 채널 1개 발견

3단계: 50→100명 (검증된 채널 집중)
├─ 방법: 2단계에서 검증된 채널에 집중 투자
├─ 채널: 검증된 1~2개 채널
├─ 목표: 리텐션 있는 사용자 100명 달성
└─ 성공 기준: D30 리텐션 목표치 달성 여부 확인
```

## 3. CAC 채널별 효율성 계산법

```
CAC(Customer Acquisition Cost) 계산:
CAC = 해당 채널 총 지출 비용 / 해당 채널로 획득한 고객 수

예시:
  Meta 광고: 100만원 투자 → 50명 가입 → CAC = 2만원
  커뮤니티: 20시간 투자(시간당 5만원 환산) → 30명 가입 → CAC = 3.3만원
  입소문: 0원 투자 → 20명 가입 → CAC = 0원

주의: 가입자 CAC와 유료 전환 고객 CAC를 분리해서 계산해야 한다.
유료 전환률 5%라면 유료 고객 CAC = 가입자 CAC × 20배
```

| 채널 | 일반적인 CAC 범위 (B2C AI) | LTV:CAC 목표 |
|---|---|---|
| 오가닉/SEO | 0~5,000원 | 5:1 이상 |
| 커뮤니티 | 0~1만원 | 5:1 이상 |
| 바이럴/WoM | 0~5,000원 | 10:1 이상 |
| SNS 인플루언서 | 5,000~3만원 | 3:1 이상 |
| 유료 광고 | 3만~20만원 | 3:1 이상 (최소) |

## 4. AI 서비스 입소문(WoM) 설계 방법

AI 서비스의 WoM은 자연 발생을 기다리는 것이 아니라 설계해야 한다.

**WoM 설계 3요소:**

1. **공유할 이유 만들기**
   - AI 결과물 자체가 공유 가능한 포맷이어야 함 (이미지, 텍스트, 영상)
   - 예: "AI가 분석한 내 발음 점수 카드" → 공유 버튼
   - 나쁜 예: AI 결과를 외부에 공유할 방법이 없음

2. **공유 마찰 제거하기**
   - 공유 버튼 1클릭, 결과물 자동 캡처, 링크 복사
   - 공유 시 "OO님의 추천으로" 추적 가능한 레퍼럴 코드

3. **공유의 사회적 보상 설계**
   - 공유자: 무료 크레딧, 구독 연장, 프리미엄 기능 잠금해제
   - 수신자: 첫 사용 특별 혜택 (온보딩 전환률 향상)

---

## 5. B2B AI 서비스 GTM 특이 사항

B2B는 B2C와 GTM 접근 방식이 다르다:

| 항목 | B2C AI | B2B AI |
|---|---|---|
| 첫 계약 방법 | 셀프 가입, 자동 온보딩 | 창업자 직접 세일즈, 파일럿 제안 |
| 의사결정자 | 개인 사용자 | 부서장, IT팀, C-레벨 (복수) |
| 세일즈 사이클 | 즉시~1일 | 2주~6개월 |
| 채널 | 검색, SNS, 바이럴 | LinkedIn, 컨퍼런스, 파트너 |
| 첫 100개 전략 | 제품 주도 성장(PLG) | 세일즈 주도 성장(SLG) |

## 체크리스트

- [ ] 첫 100명을 어디서, 어떻게 확보할 것인지 구체적으로 정의되어 있는가?
- [ ] 각 채널별 예상 CAC와 LTV:CAC 비율을 계산했는가?
- [ ] 채널 테스트를 위한 소규모 실험 계획이 있는가?
- [ ] AI 결과물이 자연스럽게 공유될 수 있는 포맷으로 설계되어 있는가?
- [ ] WoM이 발생하는 구체적인 메커니즘이 제품에 내재화되어 있는가?
- [ ] B2B라면 창업자가 직접 세일즈를 할 계획이 있는가?
- [ ] 유료 광고 이전에 오가닉 채널로 최소 검증을 계획하고 있는가?

## 흔한 레드플래그

- **"출시하면 퍼질 것이다"**: 입소문은 설계하지 않으면 발생하지 않는다. "좋은 제품은 자연스럽게 퍼진다"는 2010년대 신화다.
- **SNS 마케팅만**: 인스타그램 팔로워, 유튜브 조회수는 GTM 전략이 아니다. 가입 전환과 리텐션으로 이어지는 경로가 없으면 공허하다.
- **유료 광고 첫 번째 채널**: PMF 미달 상태에서 유료 광고를 쓰면 CAC만 높이고 리텐션 없이 이탈한다. 오가닉으로 먼저 검증하라.
- **모든 채널에 동시 투자**: 초기에 3개 이상 채널을 동시에 실행하면 어떤 채널이 효과적인지 알 수 없다.
- **GTM 채널이 타겟 사용자와 미스매치**: 50대 B2B 구매 담당자를 인스타그램 릴스로 획득하겠다는 계획.
- **앱스토어 출시 = GTM**: 앱스토어에 올리는 것은 배포다. 고객이 어떻게 발견하고 설치하고 전환하는지가 GTM이다.
